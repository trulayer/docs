---
title: "Settings — Appearance"
description: "Switch the TruLayer dashboard between light, dark, and system themes."
---

**Settings → Appearance** controls how the dashboard looks. The setting is per-browser
and saved locally — your teammates can pick a different theme without affecting yours.

## Theme modes

| Mode | What it does |
| ---- | ------------ |
| **Light** | Always use the light theme. |
| **Dark** | Always use the dark theme. |
| **System** | Follow your operating system preference. Switches automatically when your OS toggles between light and dark. |

The default for new sessions is **System**.

## Where to switch

- The sun / moon icon in the top navigation bar cycles **Light → Dark → System** with one click.
- The radio cards on **Settings → Appearance** let you pick a specific mode directly.

Both controls write to the same browser preference — switching in one place updates the other.

## Notes

- The preference is stored in your browser's local storage and is not synced across browsers or devices.
- All charts, the trace timeline, span detail panels, and code blocks render correctly in both modes.
